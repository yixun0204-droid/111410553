# Bubble Sort 氣泡排序

一個經典的氣泡排序演算法 Python 實作，包含中文註解說明。

## 演算法簡介

氣泡排序是一種簡單的排序演算法，重複遍歷陣列，比較相鄰元素並交換順序錯誤的 pair，直到陣列排序完成。

## 時間複雜度

- 最壞情況：O(n²)
- 最佳情況：O(n) — 已加入提前終止優化（`swapped` 旗標）
- 平均情況：O(n²)

## 程式特點

- 加入 `swapped` 旗標，若某一輪無交換即提前結束
- 每輪縮小比較範圍（`n - i - 1`），避免浪費
- 完整中文註解

## 執行

```bash
python bubble_sort.py
```
